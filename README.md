A privacy-first, offline Document Intelligence Platform running on Local AI (Llama 3.2).

## Features
* **Local AI Analyst:** Chat with PDFs using Ollama.
* **Citation-Aware RAG:** Answers with page citations (Source: Page 1).
* **Universal Converter:** Images & Word to PDF.
* **Editor:** Split and Merge PDFs.

## How to Run locally
1.  Install Python 3.12.
2.  Install [Ollama](https://ollama.com) and run `ollama pull llama3.2`.
3.  Install dependencies: `pip install -r requirements.txt`.
4.  Run the app: `streamlit run app.py`.
